# Lamber (ICT 项目全生命周期管理工具) - 项目当前状态

## 项目概述
本项目是一个基于 **Tauri** (Rust + Vanilla JS) 构建的桌面端应用，主要用于支撑“ICT 项目全生命周期”的文档自动化生成与业务数据管理。核心功能是通过解析与修改 OpenXML (DOCX) 格式的模板文件，实现文本和图片的动态填报，并包含智能财务反算等业务逻辑。

## 技术栈
- **后端 (Rust / Tauri)**: 负责核心业务逻辑处理、文件系统操作、Word 文档（DOCX / OpenXML）的解析与动态生成、图片 Base64 解码与注入。
- **前端 (HTML / JavaScript / Tailwind CSS)**: 负责用户界面的交互、动态表单配置、拖拽与剪贴板图片上传（Base64 编码）、以及前端状态联动。

## 核心功能模块

### 1. 自动化文档生成与填报
- 支持通过读取预设的 `.docx` 模板文件，将其解压为 XML 后进行动态内容替换，最后重新打包生成最终文档。
- **文本变量替换**: 精确识别 `{VARIABLE}` 格式的占位符，支持多行文本、智能字符串过滤处理（如：智能剔除项目名称中的“项目”并补充“服务”字样）。
- **图片注入支持**: 支持解析 Base64 格式的图片，动态修改 `word/document.xml` 和 `word/_rels/document.xml.rels`，将图片无缝嵌入到生成的 Word 文档中，避免破坏原有的 XML 结构。

### 2. ICT 项目生命周期模板支持
当前已支持及完善多个核心阶段的文档生成：
- **ICT项目立项签批表（仅适用50万以下项目）**:
  - 支持“是否垫资”选项配置。
  - 支持与其他模块（如：IT部分商务模式、资金来源、付款方式等）的前端状态数据同步联动。
  - 动态日期填充，默认当天。
- **ICT项目售前方案会审纪要**:
  - 联动立项签批表相关字段。
- **ICT项目需求导入表**:
  - 动态条件渲染：依据前端选项动态隐藏或显示“公示网址”、“信息安全评估”等特定模块。
  - 附件图片功能：支持从剪贴板粘贴（Ctrl+V）或拖拽上传图片作为客户确认函及网址公示截图。
  - 智能签字栏：当项目金额低于300万时，动态省略“分管领导”签字栏位。

### 3. 经济效益评估与智能反算
- **智能反算收入功能**: 在已知项目投入和目标净现值（NPV）/ 毛利率的情况下，系统能够自动反算所需的项目收入金额，辅助售前及财务评估。

## 近期更新与优化记录
1. **需求导入表 UI 现代化升级**: 使用 Tailwind CSS 重新设计了专属配置区，加入了折叠动画 (Collapse) 和现代化的拖拽图片上传区域 (Dropzone)。
2. **图片上传体验优化**: 增加了“清除图片”功能，允许用户在粘贴或选择图片后方便地移除和重置。
3. **修复文档生成 Bug**: 优化了 Rust 后端正则表达式的非贪婪匹配规则，彻底解决了因正则匹配导致生成空白文档及程序闪退 (Panic) 的严重问题。
4. **前端状态同步**: 完善了各表单间的联动逻辑，确保勾选框操作能正确控制后续字段及附件模块的显示与隐藏，并且在隐藏时能自动清理相关数据负载。
